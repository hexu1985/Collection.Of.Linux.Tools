VMware-ubuntu磁盘扩容及分区扩容

山山

15 人赞同了该文章

目标：硬盘原容量为100G，想扩充为400G。

条件：虚拟机所在磁盘可用容量大于400G。

步骤：

  1. VMware虚拟机设置上操作，如图

2. 进入Ubuntu，使用命令：sudo fdisk -l，查看磁盘空间，发现”Disk /dev/sda: 400GiB” 字样。然而用命令fd -h查看发现还是原来的100G。这是因为该操作仅扩展了磁盘空间，但没有分配到linux系统的分区当中。

3.使用Gparted工具进行分区扩容

（1）安装Gparted

sudo apt-get install gparted

(2) 打开Gparted

可以在应用程序里搜索Gparted打开，也可以用命令打开

sudo gparted

（3）扩展分区

列表中有unallocate的分区，是磁盘已扩容，但尚未分配的存储空间。选中列表中要扩展的分区，右键，选择"Resize/Move"。

调整New size（新的大小）和Free space following（调整后的剩余大小），完事之后点击Resize。

完事之后，点击绿色对勾，即完成。
