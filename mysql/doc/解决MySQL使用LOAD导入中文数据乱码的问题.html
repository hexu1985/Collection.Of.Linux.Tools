出现乱码的SQL语句

LOAD DATA LOCAL INFILE "employee.txt" INTO TABLE EMPLOYEE;
  • 1

解决方法，在后面添加character set utf8

LOAD DATA LOCAL INFILE "employee.txt" INTO TABLE EMPLOYEE character set utf8;
